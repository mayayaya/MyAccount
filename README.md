# Mayaの記帳系統
-web期末作業-
## 簡述功能
- 分享一些關於理財的文章和影片
- 最主要就是記帳功能，可以輸入名稱、分類、價錢、還有日期以便去做記帳，可以去計算總花費；可進行新增、修改、刪除、還有詳細看資料的功能；而重整之後資料不會消失。
- 記帳主頁的支出分類選單只是示意，並無製作跳轉連結。
- 新增支出、編輯支出的部分，在分類部分還是需要手動輸入，下拉選單只是示意(因為不知道怎麼讀下拉選單的資料到資料庫QAQ)。
